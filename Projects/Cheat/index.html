HTML & CSS Cheat Sheet

HTML Structure & Metadata

TagExampleDescription
<!DOCTYPE html>–Declares the document type. Must be first line!
<html><html lang="en">Root HTML element. Use lang for accessibility.
<head>–Metadata section (not visible on page).
<meta charset="UTF-8">–Character encoding. Always include!
<title><title>My Page</title>Page title shown on browser tab.
<link rel="stylesheet"><link href="style.css">Links external CSS file.

HTML Content Tags

TagExampleDescription
<h1> to <h6><h1>Heading</h1>Headings. Use one <h1> per page.
<p><p>Text</p>Paragraph text.
<a href=""><a href="url">Link</a>Creates a hyperlink.
<img src="" alt=""><img src="img.jpg" alt="desc">Embeds an image. Always use alt!
<br>–Line break.
<hr>–Horizontal line.

HTML Structure & Layout

TagExampleDescription
<div><div>Block</div>Block container.
<span><span>Inline</span>Inline container.
<section><section>Content</section>Semantic section of content.
<header>–Intro content or navigation.
<footer>–Footer of page or section.
<nav>–Navigation links.

HTML Forms & Inputs

TagExampleDescription
<form><form action="" method="post">Form container.
<input><input type="text">Single-line input field.
<textarea><textarea>Multiline text field.
<button><button>Click</button>Clickable button.
<label><label for="id">Label for input. Use for accessibility!

CSS Selectors & Syntax

SelectorExampleDescription
elementp {}Selects all paragraphs.
.class.note {}Selects elements with class.
#id#header {}Selects unique ID.
** {}Universal selector.
element1 element2div p {}Selects nested elements.

CSS Box Model

AreaWhat It IsDescription
ContentText or image insideMain area of the element.
PaddingSpace inside borderPushes content inward.
BorderLine around paddingVisible box edge.
MarginSpace outside borderPushes other elements away.
